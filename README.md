# oh-billy

A web-based timeline visualization application for displaying chronological events.

## Key Features
* Progressive Web App (PWA) functionality with offline support
* Interactive timeline display
* JSON-based data management for easy content updates

## Tech Stack
* HTML5
* CSS3
* JavaScript
* Service Workers
* JSON for data storage

## Usage
1. Add events to timeline-data.json
2. Deploy to a web server
3. Access via browser - supports both desktop and mobile devices
4. Can be installed as a PWA for offline access
